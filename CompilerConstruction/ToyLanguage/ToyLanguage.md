# ToyLanguage Project

This folder contains a small toy language interpreter built for Compiler Construction practice.
The language uses `dikhao("...")` as the print function, where `dikhao` means "show" in Urdu romanized form.

Files included:
- `toy_language.py`: Interpreter for the toy language.
- `example.toy`: Example script demonstrating `dikhao` and arithmetic.
- `README.md`: Project instructions and usage.

## Features
- `dikhao("Hello")` prints text to the console.
- `btao()` reads input from the user.
- Conditional blocks with `agar`, `yahtu`, and `warna`.
- Loops with `jabtak`, `chalo`, and `kro jabtak`.
- Boolean operators: `aur`, `ya`, `nahi`.
- Boolean literals: `sahi`, `galat`.
- Comparison operators: `==`, `!=`, `<`, `<=`, `>`, `>=`.
- Variables and expressions.
- Comments start with `#`.
